# Курсовой проект
course_project

## Описание проекта
Программа для анализа пересечений кривых, вычисления площади фигуры, ограниченной графиками функций, и определения количества итераций при нахождении корней. Работает с уравнениями:
- `f1 = 0.6x + 3`
- `f2 = (x-2)³ - 1`
- `f3 = 3/x`

## Структура файлов
- `main.c` - основной файл программы с точкой входа, функцией main
- `flags.c` - файл описания функций обработки ключей командной строки
- `flags.h` - файл прототипов функций обработки ключей командной строки
- `calculate.c` - файл описания функций для расчёта пересечений и площади
- `calculate.h` - файл прототипов функций для расчёта пересечений и площади
- `root.c` - файл с описанием функций для нахождения корней уравнений
- `root.h` - файл с прототипами функций для нахождения корней 
- `integral.c` - файл с описанием функций численного интегрирования
- `integral.h` - файл с прототипами функций интегрирования
- `test.c` - файл с тестами для функций нахождения корней и интегрирования
- `test.h` - файл с прототипами тестовых функций
- `makefile` - файл инструкция для утилиты сборки mingw32-make
- `readme.md` - файл с описанием проекта
- `course_project.exe` - собранный исполняемый файл программы

## Использование
Программа является консольным приложением. Рекомендуется запускать из командной строки.

### Ключи запуска
- `-help` - показать справку о доступных ключах запуска
- `-intersections` - найти и вывести точки пересечения кривых
- `-iterations` - вывести количество итераций для нахождения корней
- `-area` - вычислить площадь фигуры, ограниченной кривыми
- `-test` - запустить тесты для функций нахождения корней и интегрирования

### Примеры запуска
```bash
course_project
course_project -help
course_project -intersections
course_project -intersections -iterations
course_project -area
course_project -test
```

## Сборка
Для сборки программы выполните в PowerShell:
```bash
mingw32-make.exe clean
mingw32-make.exe
```

## Требования
MinGW-w32 (для компиляции)
PowerShell (для запуска команд сборки)
